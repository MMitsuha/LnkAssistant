## Build

1. Make sure you have [Qt](https://www.qt.io) installed
2. Install [xmake](https://xmake.io/#/).
3. Clone project and create a terminal in the folder
4. Configure the project using `xmake f --qt={YOUR_QT_PATH}`
5. Build using `xmake`
